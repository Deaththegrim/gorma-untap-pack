# Gorma, the Gullet — custom card pack for untap.in

Frog/mutant-reskinned **Gorma, the Gullet** (Golgari reanimator-aristocrats)
Commander deck. 488x680 PNG, frame + rules text + custom art, rounded corners.
Built with MTGathering's card studio.

Starting with the commander to check presentation in untap.in; the rest of the
pack follows once it looks right.

Use in untap.in **custom card game** rooms: each custom card takes a name and a
public image URL (the raw link to a file under `cards/`).

Personal, non-commercial fan content.
