# Gorma, the Gullet — custom card pack for untap.in

85 fully-rendered custom cards (Golgari reanimator-aristocrats "Gorma"
Commander deck), frog/mutant reskin, 488x680 PNG, frame + rules text + custom
art, rounded corners. Built with MTGathering's card studio.

- `index.html` — visual gallery with copy-ready image URLs
- `untap_custom_deck.txt` — the deck list, one hosted image URL per card
- `manifest.json` — machine-readable name -> URL map
- `cards/` — the card images

Use in untap.in **custom card game** rooms: each custom card takes a name and a
public image URL — use the URLs from the gallery or manifest.

Personal, non-commercial fan content.
